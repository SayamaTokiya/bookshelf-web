# 業務要件
## 仕様
- 本を登録
- 登録されてる本の情報表示
   - 申請中…まだない
   - 貸出中…誰かが借りてる
   - 保管中…今ある

- 本棚にほしい機能
   - 検索機能
   - ユーザ登録
       - 誰が借りてるのか
       - 誰がほしがってる本なのか
       - 誰が読んだことあるのか
       - その本に対する評価コメント
   - レンタル期間表示早見カレンダーページ
   - 借入予約機能
   - 本のジャンル分け機能
   - 総本数、貸出中本数、申請中本数表示
   - slack連携
       - 期限告知
       - 延滞告知
       - 借りてる人へのメッセージリンク
